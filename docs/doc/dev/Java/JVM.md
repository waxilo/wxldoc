## 运行时数据区

| 区域       | 共享  | 作用                              | GC  |
| -------- | --- | ------------------------------- | --- |
| 堆        | 共享  | 存放对象实例。                         | 会   |
| Java虚拟机栈 | 私有  | 存储调用的方法，每个方法作为一个栈帧。             | 会   |
| 本地方法栈    | 私有  | 存储调用的native方法，每个native方法作为一个栈帧。 | 会   |
| 计数器      | 私有  | 记录CPU执行位置，提供线程切换能力。             | 不会  |
| 方法区      | 共享  | 存储类元数据、运行时常量池、静态变量引用。           | 会   |
